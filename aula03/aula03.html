arrays/objetos strings

  • valueof()
    • - retorna o valor primitivod do objetos
  • charAt()
    • - retorna uma string contendo o caractere de posicao
  • concat(string, string,...)
    • - este metodo retorna uma string contendo string
  • replace
    • - busca na string uma substring que seja igual ao conteudo
  • slice(ini,fim)
    • - retornar a substring contendo os caract. inicio ate o fim
  • split(separator, limit)
    • - retorna um obj. array contendo as substring entre o seu separador e o limite
  • (inicio, fim)
    • - retorna um resultado da conversão di objeto que chamou o metodo(inicio até fim)
  • toLowerCase
    • tudo em minuscula
  • toUpperCase
    • tudo em maiuscula
