arrays/objetos strings

  • valueOf()
    • - retorna o valor primitivo do objeto
  • charAt()
    • - retorna uma string contendo o caractere de posicao
  • concat(string, string, ..., n)
    • - este método retorna uma string contendo string + stringN
  • replace
    • - busca na string uma substring para substituir por uma nova substring
  • slice(ini, fim)
    • - retorna a substring contendo os carac. inicio ate o fim
  • split(separator, limit)
    • - retorna um obj. array contendo as substring entre o separator e limite
  • substring(inicio, fim)
    • - retorna o resulta da conversão do objeto que o chamou o método (inicio até fim)
  • toLowerCase
    • tudo em minuscula
  • toUpperCase
    • tudo em maiuscula
